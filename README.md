# AI-Assisted Smart Bus Decision System 🚍

An interactive transit tracking and efficiency prototype developed for the **Puduvai Innovathon Hackathon 2026** (organized by AIC-PEC, Puducherry)[cite: 3]. The system proposes a hardware-software integrated solution aimed at modernizing public transportation infrastructure to reduce commuter uncertainty[cite: 3].

---

## 📌 Project Overview
Daily bus users often face confusion due to a lack of clear, real-time information regarding bus schedules, arrival times, and passenger density[cite: 3]. This system addresses those challenges through a combination of automated passenger counting hardware and an interactive user interface[cite: 3].

### System Architecture Details:
* **Autonomous IoT Kit:** A hardware unit utilizing an ESP32 microcontroller and IR Break-Beam sensors to automatically count passengers at both doors[cite: 3].
* **Intelligent Edge Logic:** Passenger counting algorithms designed to activate only when bus speed equals zero, preventing false counts while in motion[cite: 3].
* **User-Centric Decision Application:** A web interface providing real-time arrival estimates and live crowd density tracking to help commuters make informed decisions[cite: 3].
